Title 1- CSV to TABLE

Summary

This tool is used to convert a CSV file into a Table in a Geodatabase. The tool then uses the table to sort the data based on Time Field and creates new sorted table in the same geodatabase. Then, the tool generates a new field to be used in the Concatenation Tool later as the next step for the probability analysis.


Usage

Convert CSV to Table in Geodatabase, Sort the data, and Add field for concatenation.


Syntax

CSVtoTABLE (Input_CSV_File, Output_Geodatabase, Table_Name, Time_Field, Field_Name, Field_Length)
