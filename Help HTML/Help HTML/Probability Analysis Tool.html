Title 3- Probability Analysis

Summary

This tool is used to finally calculated the probability that a user visits country B after visiting country A. The output is a CSV file.


Usage

This tool uses the concatenate field to find the probability in percentage that a user will visit country B after visiting country A.


Syntax

ProbabilityAnalysis (Input_Table, UserID_Field, Input_Concatenated_Field, Country_A, Country_B, Output_Folder)

Parameter Explanation Data Type

The input table containing the concatenate field
